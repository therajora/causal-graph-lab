# causal-graph-lab

Repositorio inicial para estudo de grafos causais em Python, com foco em exemplos, conceitos e referencias teoricas.

## Começando

Este projeto foi inicializado com Git e uma estrutura minima para evolucao futura.

## Estrutura

- references/: notas e bibliografia
- examples/: scripts executaveis
- notebooks/: narrativa e exploracao
- src/: pacote minimo de apoio

## Instalacao (opcional)

pip install -e ".[dev,examples,notebooks]"

## Rodar testes

pytest -q
